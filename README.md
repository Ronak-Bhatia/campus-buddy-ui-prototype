Campus Buddy is a UI/UX prototype designed using Canva for a college-based
assistant application. The project focuses on helping students easily
navigate campus facilities and access important information.

## Project Type
UI / UX Prototype (No Code)

## Designed Using
- Canva

## Screens Included
- Login / Sign Up Screen
- Campus Map with Search Option
- Cafeteria Details (Menu, Timings, Seating)
- Library Details (Timings, Services, Contact)
- FAQ / Help Section
- Chatbot Concept for Student Assistance

## Key Features (Concept)
- Easy campus navigation
- Quick access to important locations
- Student-friendly interface
- Centralized help and FAQ system

## Purpose
- Visualize the idea of a campus assistant app
- Plan user flow before development
- Present UI/UX concept for learning and portfolio use

## Note
This project is a design prototype only.
No frontend or backend code has been implemented yet.
